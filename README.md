# ExpressJS with TypeScript and MVC Pattern

This project is aimed at exploring the Express.js framework with TypeScript while implementing the MVC (Model-View-Controller) pattern.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine. 
By default, the server runs on http://localhost:3000.

### Installation

   ```bash
   git clone https://github.com/vDalken/typescript-tutorial.git
   cd typescript-tutorial
   npm install
   npm run build
   npm run start
